Toyota Supra GT4

The Toyota Supra GT4 is a high-performance, track-focused version of the Toyota GR Supra.

Designed for racing, it features a stripped-down interior, upgraded suspension, and a more powerful version of the 3.0L turbocharged inline-six engine.

The Supra GT4 is built for competitive motorsport, offering exceptional handling and aerodynamics, and is intended for use in GT racing series.
